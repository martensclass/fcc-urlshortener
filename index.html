FreeCodeCamp URL Shortener

Usage

Enter a valid website address at the end of this URL starting with /new/http://www. and the server will respond with the address and a shortcut URL for that address. If the address does not conform to this format, the server will respond with an error.

            https://fcc-urlshortener-martensclass.c9users.io/http://www.google.com
            
            Will procduce:
            
            {"shortcut":https://fcc-urlshortener-martensclass.c9users.io/4eRp,"url":"http://www.google.com"}

Alternatively, you can append a previously created 4-character shortcut address to the end of this url and be redirected to that address:

            https://fcc-urlshortener-martensclass.c9users.io/4eRp

This example will redirect you to Google.com. If the shortcut does not exist or does not contain 4-characters, the server will respond with an error.
